Tradition flow of HTML elements

Elements stack on top of each other, unless they are strings, images, span and anchor tags to name a few.

Block elements start on a new line and stack on top of another.

Inline elements do not start on a new line and only take up as much width as necessary.

Inline-block elements are similar to inline elements, but they can have a width and height.

You can also assign the positionproperty one of the following values:

  • static: this follows default flow of page.
